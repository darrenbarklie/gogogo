# LEARN GO WITH TESTS

[https://quii.gitbook.io/learn-go-with-tests/](https://quii.gitbook.io/learn-go-with-tests/)

```zsh
# install godocs
go install golang.org/x/tools/cmd/godoc@latest

# launch godocs
godoc -http=localhost:6060
```

## Run

```zsh
# Run file
go run filename.go

# Run tests
go test

# Run benchmarks
go test -bench=.
```
